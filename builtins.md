# Встроенные функции

## print

**`print()`** — используется для вывода информации в консоль. Она позволяет указывать несколько аргументов, которые при выводе данных разделяются символом пробела. По умолчанию функция добавляет ровно один пробел между всеми своими аргументами.

Функция `print()` имеет два необязательных параметра:

- `sep` (separator – разделитель) позволяет установить строку, с помощью которой будут разделены аргументы (если их несколько) при печати. По умолчанию значением этого параметра является пробел — `set=' '`.

- `end` определяет строку, которая будет добавлена после вывода всех аргументов. По умолчанию значением этого параметра является символ переноса строки — `end='\n'`.

```py
print('Hello', 'World', sep='*', end='!')
Hello*World!
```

## input

**`input(n)`** — используется для ввода информации через консоль. В качестве необязательного параметра она принимает значение `n`, которое выводится в консоль перед вводом. Возвращаемый результат мы можем сохранить в переменную.

```py
name = input('Your name: ')
```

По умолчанию функция `input()` получает и возвращает данные **в формате строки**.

## round

**`round(x, n)`** — округляет число `x` до ближайшего целого. Если дробная часть числа равна 0.5, то число округляется до ближайшего четного числа. Имеет второй аргумент `n`, который позволяет указать количество знаков после запятой.


## range

**`range(start, stop, step)`** — используется, когда необходимо сгенерировать последовательность чисел. Имеет один обязательный параметр `stop` и два необязательных `start` и `step`.

Если функция получает **только один аргумент**, она создает последовательность чисел от `start` **по умолчанию равному `0`** до значения параметра `stop`, заданного при вызове. При этом необязательный параметр `step` **по умолчанию равен `1`**, поэтому последовательность чисел будет создаваться **с шагом в `1`**.

```py
for i in range(7):
    print(i, end=' ')
0 1 2 3 4 5 6
```

**Примечания:**

- Если **первый** аргумент **больше второго**, функция генерирует **пустую последовательность**.

- Чтобы сгенерировать убывающую последовательность чисел, необходимо задать в параметр `step` **отрицательное значение**.

- Параметр `step` не может равняться нулю, это вызывает ошибку.

- Если **первый** аргумент **меньше второго**, но значение параметра `step` отрицательное, функция генерирует **пустую последовательность**.

- Функция `range()` может генерировать только **целые числа, включая отрицательные**.

## sorted

**`sorted(iterable, *, key=None, reverse=False)`** — в качестве аргумента принимает итерируемый объект `iterable` и возвращает новый отсортированный список из элементов этого объекта.

```py
numbers = (5, 3, 4, 2, 1)

sorted_numbers = sorted(numbers)

print(sorted_numbers)

[1, 2, 3, 4, 5]
```

Функция `sorted()` имеет 2 необязательных параметра `reverse` и `key`. Параметр `key` задаёт функцию для извлечения ключа сравнения (например, `key=str.lower`).
Параметр `reverse=True` включает сортировку по убыванию.

Сортировка при помощи функции `sorted()` стабильна, то есть гарантирует неизменность взаиморасположения равных между собой элементов.

## map

**`map(func, iterable, *iterables)`** — возвращает [итератор](glossary.md#итератор) с результатами применения функции `func` к элементам итерируемого объекта `iterable`. Если передано несколько итерируемых объектов, в функцию `func` будут передаваться сразу несколько элементов, расположенных на одинаковых позициях.

```py
lst1 = [1, 2, 3, 4, 5]
lst2 = [10, 20, 30, 40, 50]

print(list(map(lambda a, b: a + b, lst1, lst2)))

[11, 22, 33, 44, 55]
```

Функция `map()` возвращает итератор, чтобы получить из него список, мы преобразовали его с помощью функции `list()`.

## filter

**`filter(func, iterable)`** — возвращает [итератор](glossary.md#итератор), содержащий только те элементы итерируемого объекта `iterable`, для которых функция `func` возвращает `True`. Если вместо `func` указать `None`, каждый элемент будет проверен на соответствие значению `True`.

```py
lst = ['a', '', 0, 32, True, '12', 1, False]

print(list(filter(lambda s: isinstance(s, str), lst)))
print(list(filter(None, lst)))

['a', '', '12']
['a', 32, True, '12', 1]
```

Функция `filter()` возвращает итератор, чтобы получить из него список, мы преобразовали его с помощью функции `list()`.

## reduce

**`reduce(func, iterable, initializer=None)`** — последовательно применяет функцию `func` к элементам итерируемого объекта `iterable` слева направо, сводя его к одному значению. Начальное значение задаётся через `initializer`. Если начальное значение не установлено, то в его качестве используется первое значение `iterable`.

```py
from functools import reduce

def mul(x, y):
    return x * y

lst = [3, 7, 11, 5]

print(reduce(mul, lst))

1155
```

Функция `reduce()` во второй версии языка Python была встроенной, но в Python 3 ее решили перенести в модуль `functools`.

## all

**`all(iterable)`** — возвращает значение `True`, если **все элементы** переданного ей итерируемого объекта **истинны** (приводятся к значению `True`), или `False` в противном случае.

```py
lst1 = [True, 1, '2', (3,), [4], 5.0]
lst2 = [True, 1, '2', (3,), [4], 5.0, False]
lst3 = []

print(all(lst1), all(lst2), all(lst3), sep='\n')

True
False
True
```

---

## any

**`any(iterable)`** — возвращает значение `True`, если **хотя бы один элемент** переданного ей итерируемого объекта является **истинным** (приводится к значению `True`), или `False` в противном случае.

```py
lst1 = [True, 0, '', [], False]
lst2 = [False, 0, '', []]
lst3 = []

print(any(lst1), any(lst2), any(lst3), sep='\n')

True
False
False
```

## enumerate

**`enumerate(iterable, start)`** — возвращает кортеж из индекса элемента и самого элемента переданного ей итерируемого объекта. С помощью необязательного параметра `start`(по умолчанию равному `0`) можно задать начальное значение индекса.

```py
lst = ['one', 'two', 'three']

print(list(enumerate(lst)))

[(0, 'one'), (1, 'two'), (2, 'three')]
```

Функция `enumerate()` возвращает итератор, чтобы получить из него список, мы преобразовали его с помощью функции `list()`.

## zip

**`zip(iter1, iter2)`** — позволяет параллельно **объединить несколько итерируемых объектов**. Возвращает **итератор кортежей**, где:

- первый кортеж содержит первые элементы всех объектов,
- второй — вторые элементы,
- и т.д

Например объединим 2 списка в словарь:

```py
lst1 = [1, 2, 3]
lst2 = ['one', 'two', 'three']

result = zip(lst1, lst2)

print(result) # выводим итератор
print(dict(result)) # преобразуем в словарь

<zip object at 0x72ca8b721ac0> # итератор
{1: 'one', 2: 'two', 3: 'three'} # словарь
```

Обратная операция — распаковка через `zip(*iterable)`:

```py
pairs = [(1, 'a'), (2, 'b'), (3, 'c')]
nums, letters = zip(*pairs)

print(nums, letters, sep = '\n')

(1, 2, 3)
('a', 'b', 'c')
```

**Примечания:**

- `zip()` завершает работу, когда заканчивается самый короткий из переданных итерируемых объектов.
- Работает с любым количеством аргументов.
- Возвращает итератор.

## ord

**`ord()`** — позволяет **определить код символа** в таблице символов `Unicode`.

```py
print(ord('A'))

65
```

**Примечания:**

- Функция `ord()` принимает только **одиночный символ**. Если аргументом функции передать строку из нескольких символов, **результатом будет ошибка времени выполнения**.

## float

**`float()`** — используется для преобразования в число с плавающей точкой.

**Преобразование чисел** с плавающей точкой в целое производится **с округлением в сторону нуля**, то есть `int(1.7) = 1`, `int(-1.7) = -1`.

Если строка не является **целым числом**, то при преобразовании возникнет **ошибка**.

## help

**`help()`** — встроенная функция для вывода интерактивной справки об объектах, модулях, методах и ключевых словах Python. Она отображает содержимое атрибута `__doc__`, а также структуру объекта (его методы и атрибуты).

**Примечания:**

- Если у объекта нет [docstrings](functions.md#docstrings--строки-документации), `help()` всё равно покажет техническую информацию (например, список доступных методов через `dir()`).
- `help()` умеет работать с строковыми запросами, например `help("modules")`, чтобы показать список всех установленных библиотек.

## dir

**`dir()`** — возвращает список имён, определяемых объектом. Например, для модуля в этот список входят функции, классы и переменные, определённые в этом модуле. Также мы можем выполнить `dir()` для какого нибудь другого объекта, класса или функции, чтобы узнать его атрибуты.

```py
dir(str) # выведет список атрибутов класса
```

Если вызвать функцию `dir()` без аргументов, она вернет список имен определенных в текущем модуле.
