# Часто используемые функции

**`print()`** — используется для вывода информации в консоль. Она позволяет указывать несколько аргументов, которые при выводе данных разделяются символом пробела. По умолчанию функция добавляет ровно один пробел между всеми своими аргументами.

Функция `print()` имеет два необязательных параметра:

- `sep` (separator – разделитель) позволяет установить строку, с помощью которой будут разделены аргументы (если их несколько) при печати. По умолчанию значением этого параметра является пробел — `set=' '`.

- `end` определяет строку, которая будет добавлена после вывода всех аргументов. По умолчанию значением этого параметра является символ переноса строки — `end='\n'`.

```py
print('Hello', 'World', sep='*', end='!')
Hello*World!
```

---

**`input()`** - используется для ввода информации через консоль. В качестве необязательного параметра она принимает значение, которое выводится в консоль перед вводом. Возвращаемый результат мы можем сохранить в переменную.

```py
name = input('Your name: ')
```

По умолчанию функция `input()` получает и возвращает данные **в формате строки**.

---

**`len()`** —  возвращает длину строки (число символов, из которых состоит строка).

```py
print(len('Hello'))
5
```

---

**`type()`** - позволяет динамически узнать текущий тип переменной.

```py
num1 = int(input())
print(type(num1))
<class 'int'> # получаем число
```

---

**`max()`** — используется для определения максимального числа из заданных аргументов.

```py
print(max(1,2.4,3))
3
```

---

**`min()`** — используется для определения минимального числа из заданных аргументов.

```py
print(min(1,2.4,3))
1
```

---

**`abs()`** — используется для нахождения модуля(абсолютной величины) числа.

```py
print(abs(-7))
7
```

---

# Функции преобразования типов

**`int()`** — используется для преобразования к целочисленному типу данных `integer`.

---

**`str()`** — используется для преобразования в строковой тип данных `string`.

---

**`float()`** — используется для преобразования в число с плавающей точкой.

**Преобразование чисел** с плавающей точкой в целое производится **с округлением в сторону нуля**, то есть `int(1.7) = 1`, `int(-1.7) = -1`.

Если строка не является **целым числом**, то при преобразовании возникнет **ошибка**.
