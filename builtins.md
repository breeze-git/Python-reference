# Часто используемые функции

**`print()`** — используется для вывода информации в консоль. Она позволяет указывать несколько аргументов, которые при выводе данных разделяются символом пробела. По умолчанию функция добавляет ровно один пробел между всеми своими аргументами.

Функция `print()` имеет два необязательных параметра:

- `sep` (separator – разделитель) позволяет установить строку, с помощью которой будут разделены аргументы (если их несколько) при печати. По умолчанию значением этого параметра является пробел — `set=' '`.

- `end` определяет строку, которая будет добавлена после вывода всех аргументов. По умолчанию значением этого параметра является символ переноса строки — `end='\n'`.

```py
print('Hello', 'World', sep='*', end='!')
Hello*World!
```

---

**`input(n)`** — используется для ввода информации через консоль. В качестве необязательного параметра она принимает значение `n`, которое выводится в консоль перед вводом. Возвращаемый результат мы можем сохранить в переменную.

```py
name = input('Your name: ')
```

По умолчанию функция `input()` получает и возвращает данные **в формате строки**.

---

**`len()`** —  возвращает длину строки или списка.

```py
print(len('Hello'))
5
```

---

**`type()`** — позволяет динамически узнать текущий тип переменной.

```py
num1 = int(input())
print(type(num1))
<class 'int'> # получаем число
```

---

**`max()`** — используется для определения максимального числа из заданных аргументов.

```py
print(max(1,2.4,3))
3
```

---

**`min()`** — используется для определения минимального числа из заданных аргументов.

```py
print(min(1,2.4,3))
1
```

---

**`abs()`** — используется для нахождения модуля(абсолютной величины) числа.

```py
print(abs(-7))
7
```

---

**`sum()`** — используется для вычисления **суммы чисел заданной последовательности**.

```py
print(sum([1, 2, 3, 4]))
10
```

---

**`round(x, n)`** — округляет число `x` до ближайшего целого. Если дробная часть числа равна 0.5, то число округляется до ближайшего четного числа. Имеет второй аргумент `n`, который позволяет указать количество знаков после запятой.

```py
print(round(10.237298, 3))
10.237
```

---

**`range(start, stop, step)`** — используется, когда необходимо сгенерировать последовательность чисел. Имеет один обязательный параметр `stop` и два необязательных `start` и `step`.

Если функция получает **только один аргумент**, она создает последовательность чисел от `start` **по умолчанию равному `0`** до значения параметра `stop`, заданного при вызове. При этом необязательный параметр `step` **по умолчанию равен `1`**, поэтому последовательность чисел будет создаваться **с шагом в `1`**.

```py
for i in range(7):
    print(i, end=' ')
0 1 2 3 4 5 6
```

**Примечания:**

- Если **первый** аргумент **больше второго**, функция генерирует **пустую последовательность**.

- Чтобы сгенерировать убывающую последовательность чисел, необходимо задать в параметр `step` **отрицательное значение**.

- Параметр `step` не может равняться нулю, это вызывает ошибку.

- Если **первый** аргумент **меньше второго**, но значение параметра `step` отрицательное, функция генерирует **пустую последовательность**.

- Функция `range()` может генерировать только **целые числа, включая отрицательные**.

---

**`ord()`** — позволяет **определить код символа** в таблице символов `Unicode`.

```py
print(ord('A'))

65
```

**Примечания:**

- Функция `ord()` принимает только **одиночный символ**. Если аргументом функции передать строку из нескольких символов, **результатом будет ошибка времени выполнения**.

---

**`chr()`** — позволяет **получить символ**  по его коду в таблице `Unicode`.

```py
print(chr(65))

A
```

# Функции преобразования типов

**`int()`** — используется для преобразования к целочисленному типу данных `integer`.

---

**`str()`** — используется для преобразования в строковой тип данных `string`.

---

**`float()`** — используется для преобразования в число с плавающей точкой.

**Преобразование чисел** с плавающей точкой в целое производится **с округлением в сторону нуля**, то есть `int(1.7) = 1`, `int(-1.7) = -1`.

Если строка не является **целым числом**, то при преобразовании возникнет **ошибка**.

---

**Функция `list()`** — используется для преобразования в список.
