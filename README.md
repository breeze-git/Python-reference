# Python Reference (Справочник)

Краткий справочник по языку программирования Python, структурированный по темам.

Проект предназначен для быстрого повторения, навигации и закрепления базовых понятий языка.

🎯 Я создаю его для себя, чтобы быстро освежать знания, и для тех, кто только начинает свой путь в программировании.

>Важно: Это не замена официальной документации, а удобная «шпаргалка» для быстрой навигации по темам языка.

---

⚡️ Поиск по репозиторию в
GitHub позволяет быстро находить информацию в этом справочнике.

Используйте горячие клавиши:

- ⌨️ Клавиша `T` — быстрый поиск по названиям файлов. Просто нажмите `T` и начните печатать тему.
- ⌨️ Клавиша `/` — глобальный поиск по тексту внутри всех файлов справочника. Поможет найти конкретный метод или ключевое слово.
- ⌨️ Клавиша `.`  — открывает этот репозиторий в Web Editor (похоже на VS Code). Удобно, если нужно просмотреть много файлов сразу.

Также в качестве ориентира удобно использовать оглавление справочника.
