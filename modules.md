# Что такое модуль?

**Модуль** — в Python модулем называется библиотека функций, которые уже реализованы и готовы к использованию.

**Инструкция `import`** - инструкция, позволяющая подключить модуль и использовать его функции.

```py
import module # module - название модуля
```

Для того, чтобы **вызвать функцию** из модуля необходимо **обратиться к модулю** и указать название функции **через точку**:

```py
import module # импортируем модуль
module.func() # вызов функции
```

---

**В Python можно ипортировать функции из модулей и другими способами.**

**`from ... import *`** - импортирует все функции из заданного в просранство именн программы. Это позволяет вызывать все функции модуля без обращнения к модулю.

```py
from module import *
func() # можем вызвать функцию напрямую без указания модуля
```

Рекомендуется **избегать** импортирования через `from ... import *`, так как нет ясности, какие функции были импортированы и это **загрязняет пространство имён** лишними, неиспользуемыми функциями.

---

**`from ... import func_1, func_2`** - позволяет импортировать **конкретные** функции из заданного модуля в просранство именн программы. Это так же позволяет вызывать эти функции без обращнения к их модулю, однаком мы не можем использовать другие функции модуля, если они не указаны в импорте.
  
  ```py
  from module import func_1, func_2
  func_1()
  func_2()
  ```

# Полезные модули

## Модуль math

**Модуль math** — модуль, предоставляющий обширный функционал для проведения вычислений с действительными числами.

**Список основных функций модуля math:**

|Функция          |Описание                                             |
|-----------------|-----------------------------------------------------|
|**Округления**   |                                                     |
| `floor(x)`      |Округляет `x` в сторону меньшего целого числа        |
| `ceil(x)`       |Округляет `x` в сторону большего целого числа        |
|**Корни, логарифмы, степени и факториал**|                             |
| `sqrt(x)`       |Квадратный корень числа `x`                          |
| `pow(x, n)`     |Возведение числа `x` в степень `n`. Быстро обрабатывает операции с дробными числами.|
| `log(x)`        |Натуральный логарифм числа `x`. Основание — число `e`|
| `log10(x)`      |Десятичный логарифм числа `x`. Основание — число `10`|
| `log(x, b)`     |Логарифм числа `x` по основанию `b`                  |
| `factorial(n)`  |Факториал натурального числа `n`                     |
|**Тригонометрия**|                                                     |
| `degrees(x)`    |Преобразует угол `x` из радиан в градусы             |
| `radians(x)`    |Преобразует угол `x` из градусов в радианы           |
| `cos(x)`        |Косинус угла `x` (в радианах)                        |
| `sin(x)`        |Синус угла `x` (в радианах)                          |
| `tan(x)`        |Тангенс угла `x` (в радианах)                        |
| `acos(x)`       |Угол в радианах от `0` до `π`, `cos` которого равен `x`|
| `asin(x)`       |Угол в радианах от `−π/2` до `π/2,` `sin` которого равен `x`|
| `atan(x)`       |Угол в радианах от `−π/2` до `π/2`, `tan` которого равен `x`|
| `atan2(y, x)`   |Полярный угол (в радианах) точки с координатами (`x`, `y`)|
|**Константы**    |                                                     |
|`pi`             |Число `π` = `3.141592653589793`                      |
|`e`              |Число `e` = `2.718281828459045`(константа Эйлера)    |
