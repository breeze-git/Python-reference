# Тип данных `String`

**<b id="#string-type">String (str)</b>** — тип данных, значением которого являются **строки** — произвольные последовательности символов.

Значения типа String могут содержать **любые символы**, включая буквы, цифры, знаки препинания и специальные символы, и заключаются в **одинарные(' ')** или **двойные(" ")** кавычки. В Python строки представляют собой набор символов **в кодировке Unicode.**

**Для создания строковой переменной (литерала)** нужно заключить необходимый текст в **кавычки**.

```py
s = "Hello World!"
print(s)

Hello World!
```

Чтобы записать в переменнную многострочный текст, нужно заключить его в **тройные кавычки (''' ''').**

```py
longText = '''The evening sunset painted the sky
with shades of orange and pink.'''
print(longText)

The evening sunset painted the sky
with shades of orange and pink.
```

# Индексация строк. Срезы

<b id="#string-indexing-slices"></b>

<b id="#string-indexing">Индекс</b> - **порядковый номер** символа, находящегося в **составе строки**.

В Python **индексация начинается с** `0`. То есть первый символ в строке имеет индекс `[0]`, второй `[1]` и т.д.

```py
s = 'Python'
print(s[0], s[3])

P h
```

В Python также есть возможность работы с **отрицательными индексами**. Если **первый** символ строки имеет **индекс `0`**, то **последнему** элементу присваивается **индекс `-1`**.

```py
s = 'Python'
print(s[-1], s[-2])

n o
```

Таким образом **отрицательный индекс первого символа** строки будет числом всех символов строки **со знаком `-`**.

```py
s = 'Python'
print(s[-6], s[-5])

P y
```

**Примечания:**

- Обращение по несуществующему индексу **приводит к ошибке**.
- Строки являются **неизменяемыми структурами** данных. Любое изменение строки, создает **новую строку, но не изменяет старую**.

---

**Срезы (slices)** в Python — это способ **извлекать подстроку** из строки с помощью специального синтаксиса: `s[start:end:step]`.

```py
s = 'Python'
print(s[0:3:1])

Pyt
```

В примере выше `start = 0`, `end = 3`, `step = 1`.

- `start` — индекс, с которого начинается срез (включительно). По умолчанию `0`.
- `end` — индекс, на котором срез заканчивается (невключительно).
- `step` — шаг, с которым берутся символы. По умолчанию 1.

**Способы создания срезов:**

|Способ      | Пояснение                                              |
|------------|--------------------------------------------------------|
| `s[a:n]`   | строка состоящая из символов с индексами от `a` до `n` |
| `s[:n]`    | символы от начала строки до `n`(невключительно)        |
| `s[n:]`    | символы от индекса `n`(включительно) до конца строки   |
| `s[-n:]`   | символы от `-n` до конца строки                        |
| `s[:]`     | вся строка целиком                                     |
| `s[a:n:z]` | символы от индекса `a` до `n` с шагом `z`              |
| `s[::-1]`  | строка в обратном порядке, так как шаг отрицательный   |

**Примечания:**

- Если `start` **больше** `end`, срез создает пустую строку.

# Оператор `in` и итерирование строк

**Оператор принадлежности in** — позволяет проверить, **находится ли указанная строка в составе другой строки**.

```py
s = 'Python'
print('Py' in s)

True
```

С помощью цикла `for` и оператора `in` мы можем **итерировать строки**.

```py
s = 'Python'
for i in s:
    print(i, end = ' ')

P y t h o n
```

Таже мы можем перебирать символы по индексам, используя *<a href="builtins.md#len">функцию `len()`</a>*, чтобы узнать длину строки.

```py
s = 'Python'

for i in range(len(s)):
    print(s[i], end = ' ')

P y t h o n
```

# Управляющие последовательности

Для управления выводом текста и форматирования данных в Python используются **управляющие последовательности**.

Они позволяют вставлять специальные символы, которые изменяют отображение строки, например, для переноса строки, табуляции или других эффектов.

Вот некоторые из них:

| Символ | Описание                                 |
|--------|------------------------------------------|
| `\\`   | добавляет в строку **обратный слеш**     |
| `\'`   | добавляет в строку **одинарную кавычку** |
| `\"`   | добавляет в строку **двойную кавычку**   |
| `\n`   | осуществляет **переход на новую строку** |
| `\t`   | добавляет **табуляцию** (4 пробела)      |

**Чтобы отключить интерпретацию управляющих последовательностей** в строке, перед ней ставится **символ `r`**.

# Форматирование строк

## Метод `format()`

**`s.format()`** — метод класса `str`, позволяющий форматировать строки. С его помощью можно вставлять значения других переменных в исходную строку `s`.

**Например:**

```py
firstname = 'Will'
lastname = 'Turner'

print('Fullname : {0} {1}'.format(firstname, lastname))

Fullname : Will Turner
```

В примере, выше **вместо фигурных скобок** `{0}` и `{1}`, в строку **вставляются значения переменных** `firstname` и `lastname` в соответсвтующем порядке.

## f-строки

<b id="#f-strings"></b>

Есть более простой и интуитивно понятный способ форматирования строк в Python, появившийся в версии Python 3.6.

Теперь чтобы вставить в строку **значения других переменных**, нам достаточно разместить название переменных в фигурных скобках `{ }` в строку, и перед всей строкой поставить **символ `f`**.

```py
firstname = 'Will'
lastname = 'Turner'

print(f'Fullname : {firstname} {lastname}')

Fullname : Will Turner
```

Также `f-строки` обеспечивают более **высокую скорость форматирования**, чем метод `s.format` и некоторые другие способы форматирования строк в Python.

# Конкатенация и умножение строки на число

**Конкатенация** — операция, позволяющая **складывать строки** с помощью оператора сложения — `+`. Результатом такой операции является новая строка, созданная из нескольких других.

```py
print('Py' + 'thon')

Python
```

---

**Операция умножения строки на число** - позволяет умножить строку на целое неотрицательное число, при помощи оператора умножения — `*`, который повторяет строку указанное количество раз.

```py
print('Python' * 3)

PythonPythonPython
```

**При умножении строки на 0 получается пустая строка — `''`**.

# ASCII и Unicode

## Таблицы символов

**Таблица символов `ASCII`** — это набор из 128 цифровых кодов, которые обозначают английские буквы, различные знаки препинания и другие символы.

Например, код `ASCII` для прописной латинской буквы `А` — `65`. Когда на компьютерной клавиатуре вы набираете букву `А` в верхнем регистре, в памяти сохраняется число `65` в **двоичной системе** счисления.

`A = 65 = 01000001`

---

**Таблица символов `Unicode`** — это набор цифровых символов, которые включают в себя знаки почти всех письменных языков мира.

Это широкая схема кодирования, совместимая с `ASCII`. Первые 128 кодов таблицы символов Unicode совпадают с ASCII.

То, как символы с соответствующими кодами будут храниться в памяти компьютера, зависит от конкретной кодировки, базирующейся на `Unicode`,  например `UTF-8`.

---

**В Python строки хранятся в виде последовательности `Unicode` символов.**

## Функции `ord()` и `chr()`

**Функция `ord()`** — позволяет **определить код символа** в таблице символов Unicode.

```py
print(ord('A'))

65
```

Функция `ord()` принимает только **одиночный символ**. Если аргументом функции передать строку из нескольких символов, **результатом будет ошибка времени выполнения**.

---

**Функция `chr()`** — позволяет **получить символ** по его коду в таблице `Unicode`.

```py
print(chr(65))

A
```

# Сравнение строк

В Python сравнение строк происходит на основе **лексикографического порядка** – в соответствии с кодами составляющих их символов в **таблице Unicode**.

Символ `'A'` в сравнении с символом `'B'` будет **меньше**, потому что код символа `'A'` **равен** `65`, а код символа `'B'` **равен** `66`.

```py
print('A' < 'B', ord('A'), ord('B'))

True 65 66
```

**Алгоритм сравнения строк в Python:**

1. Сравниваются **первые символы** каждой строки. Если символы равны, переход к следующей паре символов.
2. Когда находится первый отличающийся символ, **строка с меньшим символом (по кодировке `Unicode`) считается `меньше`**.
3. Если одна из строк **заканчивается раньше**  другой, то она считается  **`меньшей`**.

**Примеры сравнения строк:**

```py
print('abc' < 'abz',    # c = 67, z = 122
      'abcd' < 'abcde', # первая строка короче
      'ABC' < 'abc',    # строчные символы по таблице ниже заглавных
      'abc' == 'abc')   # строки равны

True True True True
```

**Примечания:**

- **Python сравнение останавливается**, как только находится **первое различие между символами** на соответствующих позициях. Дальнейшее сравнение символов не требуется. Если **первый символ первой строки больше**, чем первый символ второй строки, то **первая строка будет больше** второй.

- Мы можем сравнивать строки, состоящие из **любых символов, которые входят в таблицу Unicode**.

- В Python **не поддерживается** операция **сравнения строк и чисел** друг с другом.

## Конвертация регистра

**`s.capitalize()`** — возвращает копию строки `s`, в которой **первый символ имеет верхний регистр**, а все **остальные символы имеют нижний регистр.**

```py
s = 'python'
print(s.capitalize())

Python
```

---

**`s.swapcase()`** — возвращает копию строки `s`, в которой все **символы верхнего регистра, преобразуются в нижний, а все символы нижнего в верхний**.

```py
s = 'PyThoN'
print(s.swapcase())

pYtHOn
```

---

**`s.title()`** — возвращает копию строки `s`, в которой **первый символ каждого слова переводится в верхний регистр**.

```py
s = 'the evening sunset painted the sky'
print(s.title())

The Evening Sunset Painted The Sky
```

---

**`s.lower()`** — возвращает копию строки `s`, в нижнем регистре.

```py
s = 'HELLO WORLD!'
print(s.lower())

hello world!
```

---

**`s.upper()`** — возвращает копию строки `s`, в верхнем регистре.

```py
s = 'hello world!'
print(s.upper())

HELLO WORLD!
```

## Поиск и замена

**`s.count(sub, start, end)`** — возвращает **количество непересекающихся вхождений** подстроки `sub` в исходную строку s.

```py
s = 'the evening sunset painted the sky'
print(s.count('the'))

2
```

Необязательные параметры `start` и `end` **задают диапазон индексов**, в пределах которого выполняется операция. По умолчанию используется **вся строка**.

---

**`s.startswith(sub, start, end)`** — возвращает значение `True`, если исходная строка **начинается с подстроки `sub`**, или `False` в противном случае.

```py
s = 'Python'
print(s.startswith('Py'))

True
```

Необязательные параметры `start` и `end` **задают диапазон индексов**, в пределах которого выполняется операция. По умолчанию используется **вся строка**.

---

**`s.endswith(sub, start, end)`** — возвращает значение `True`, если исходная строка **оканчивается на подстроку `sub`**, или `False` в противном случае.

```py
s = 'Python'
print(s.endswith('on'))

True
```

Необязательные параметры `start` и `end` **задают диапазон индексов**, в пределах которого выполняется операция. По умолчанию используется **вся строка**.

---

**`s.find(sub,start, end)`** — находит индекс **первого вхождения** подстроки `sub` в исходной строке `s`. Если строка `s` не содержит подстроки `sub`, то метод **возвращает значение `-1`**.

```py
s = 'the evening sunset painted the sky'
print(s.find('the'))

0
```

Необязательные параметры `start` и `end` **задают диапазон индексов**, в пределах которого выполняется операция. По умолчанию используется **вся строка**.

---

**`s.rfind(sub, start, end)`** — находит индекс **первого вхождения** подстроки `sub` в исходной строке `s`, **начиная с конца строки**. Если строка `s` не содержит подстроки `sub`, то метод **возвращает значение `-1`**.

```py
s = 'the evening sunset painted the sky'
print(s.rfind('the'))

27
```

Необязательные параметры `start` и `end` **задают диапазон индексов**, в пределах которого выполняется операция. По умолчанию используется **вся строка**.

---

**`s.index(sub, start, end)`** — идентичен методу `find(sub, start, end)`, находит индекс **первого вхождения** подстроки `sub` в исходной строке `s`. Но если строка `s` не содержит подстроки `sub`, то метод **вызывает ошибку `ValueError: substring not found`**.

```py
s = 'the evening sunset painted the sky'
print(s.index('the'))

0
```

Необязательные параметры `start` и `end` **задают диапазон индексов**, в пределах которого выполняется операция. По умолчанию используется **вся строка**.

---

**`s.rindex(sub, start, end)`** — идентичен методу `rfind(sub, start, end)`, находит индекс **первого вхождения** подстроки `sub` в исходной строке `s`, **начиная с конца строки**. Но если строка `s` не содержит подстроки `sub`, то метод **вызывает ошибку `ValueError: substring not found`**.

```py
s = 'the evening sunset painted the sky'
print(s.rindex('the'))

27
```

Необязательные параметры `start` и `end` **задают диапазон индексов**, в пределах которого выполняется операция. По умолчанию используется **вся строка**.

---

**`s.strip()`** —  возвращает копию строки `s`, у которой удалены все пробелы, стоящие **в начале и конце строки**.

Метод может принимать на вход необязательный аргумент `chars`. Необязательный аргумент `chars` — это строка, которая **определяет набор символов для удаления**. По умолчанию значением этого аргумента является `' '`.

**`s.rstrip()`** — возвращает копию строки `s`, у которой удалены все пробелы, стоящие **в конце строки**.

```py

```

Метод может принимать на вход необязательный аргумент `chars`. Необязательный аргумент `chars` — это строка, которая **определяет набор символов для удаления**. По умолчанию значением этого аргумента является `' '`.

## Классификация символов

**`s.isalnum()`** ­— возвращает значение `True`, если исходная **строка является непустой и состоит только из буквенно-цифровых символов**, или `False` в противном случае.

```py
s = '123Abc'
print(s.isalnum())

True
```

---

**`s.isalpha()`** — возвращает значение `True`, если исходная **строка является непустой и состоит только из буквенных символов**, или `False` в противном случае.

```py
s = 'Abc'
print(s.isalpha())

True
```

---

**`s.isdigit()`** — возвращает значение `True`, если исходная **строка является непустой и состоит только из цифровых символов**, или `False` в противном случае.

```py
s = '123'
print(s.isdigit())

True
```

---

**`s.islower()`** — возвращает значение `True`, если **все буквенные символы исходной строки являются строчными**, или `False` в противном случае.

```py
s = 'abc123_@'
print(s.islower())

True
```

---

**`s.isupper()`** — возвращает значение `True`, если **все буквенные символы исходной строки являются заглавными**, или `False` в противном случае.

```py
s = 'ABC123_@'
print(s.isupper())

True
```

---

**`s.isspace()`** — возвращает значение `True`, если **строка состоит только из пробельных символов**, или `False` в противном случае.

```py
s = '     '
print(s.isspace())

True
```

---

**`s.ljust(width[, fillchar])`** — **выравнивает строку по левому краю**, добавляя символы `fillchar` справа, пока общая длина не станет `width`.

```py
s = "Hello"

print(s.rjust(20, '-'))

Hello---------------
```

---

**`s.rjust(width[, fillchar])`** — **выравнивает строку по правому краю**, добавляя символы `fillchar` слева, пока общая длина не станет `width`. По умолчанию `fillchar` равен символу пробела `' '`.

```py
s = "Hello"

print(s.rjust(20, '-'))

---------------Hello
```

---

**`s.center(width[, fillchar])`** — **выравнивает по центру**, добавляя символы `fillchar` равномерно слева и справа, если количество нечётное добавляет лишний символ справа.

```py
s = "Hello"

print(s.center(20, '-'))

-------Hello--------
```

## Операции с другими структурами данных

**`s.split(sep=None, maxsplit=-1)`** — **разделяет строку** на список подстрок **по указанному разделителю `sep`**.

- Если `sep` не указан или равен `None`, строка разбивается по любому количеству **пробельных символов**.
- Если `sep` задан, строка разбивается **строго по этому разделителю**.
- Параметр `maxsplit` ограничивает **максимальное число разбиений** (по умолчанию — все).

```py
s = "apple,banana,cherry"

print(s.split(","))

['apple', 'banana', 'cherry']
```

---

**`s.join(iterable)`** — **объединяет элементы** итерируемого объекта `iterable` (обычно списка или кортежа строк) в **одну строку**, вставляя между ними текущую строку `s` как **разделитель**.

```py
l = ['apple', 'banana', 'cherry']

s = ' '.join(l)

print(s)

apple banana cherry
```

**Примечания:**

- Все элементы `iterable` должны быть **строками**, иначе будет **ошибка `TypeError`**.
