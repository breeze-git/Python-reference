# Условные конструкции

## Условные операторы

**Условный оператор `if-else`** — **это составная инструкция**, позволяющая производить проверку условия и управлять ходом выполнения программы.  

**Составная инструкция `if-else`** состоит из условия и двух *[блоков кода](glossary.md#блок-кода)*: один выполняется, если условие **истинно**, другой — если **ложно**.

```py
password = input('Password: ')

if password == 'qwerty':
    print('Access allowed')
else:
    print('Wrong password')
```

## Вложенные и каскадные условия

**Вложенное ветвление** — это конструкция, при которой внутри блока одного условного оператора размещаются другие условные операторы. Каждый новый уровень вложенности выделяется дополнительным отступом.

```py
num = int(input())

if num > 0:
    print('Число положительное')
else:
    if num < 0:
        print('Число отрицательное')
    else:
        if num == 0:
            print('Число равно нулю')
```

---

**Каскадный условный оператор `if-elif-else`** - оператор, который используется при проверке нескольких условий.

```py
num = int(input())

if num > 0:
    print('Число положительное')
elif num < 0:
    print('Число отрицательное')
else: 
    print('Число равно нулю')
```

**Ключевая особенность** каскадного условного оператора `if-elif-else` заключается в том, что при нахождении первого **истинного** условия его блок кода **выполняется**, а остальная часть конструкции **игнорируется**.

## Тернарный условный оператор

**Тернарный условный оператор** — это лаконичный способ записать простое условие `if-else` в одну строку.

**Синтаксис:**

`значение1 if условие else значение2`

Если условие истинно, возвращается `значение1` , если нет – `значение2`.

**Применение:**

- Для простых, понятных условий.
- В выражениях, где нужен результат (присваивание, возврат, аргумент функции).
- В генераторах и списочных выражениях.

**Пример в списочном выражении:**

```py
numbers = [1, 2, 3, 4, 5]

parity = ["even" if x % 2 == 0 else "odd" for x in numbers]

print(parity)

['odd', 'even', 'odd', 'even', 'odd']
```

**Пример в f-строке:**

```py
name = "Will"
greeting = f"Hello, {name}!" if name else "Hello, pirate!"
```

## Конструкция match-case

**Конструкция match-case (Structural Pattern Matching),** — это полноценный инструмент структурного сопоставления шаблонов, появившийся в Python 3.10. Он позволяет не только сравнивать значения, но и проверять структуру объектов, типы данных и даже [распаковывать](operators.md#операции-распаковки-и-упаковки-итерируемых-объектов-оператор--в-распаковке-и-упаковке) значения прямо в ходе проверки.

**Основной синтаксис:**

```py
match объект:
    case шаблон_1:
        # код
    case шаблон_2:
        # код
    case _: # аналог else
        # код
```

**Ключевые возможности:**

- **Сопоставление литералов** — сравнение с конкретными значениями (числа, строки, `True`, `False`, `None`);

- **Группировка шаблонов (|)** — можно объединить несколько вариантов в один `case` (логическое `ИЛИ`);

- **Распаковка структур** — `match` может проверить, является ли объект списком или кортежем определенной длины, и сразу присвоить его элементы переменным:

    ```py
    match point:
        case (0, 0): print("Центр")
        case (x, y): print(f"Координаты: {x}, {y}") 
    ```

- **Упаковка в списки и словари** — `match` позволяет упаковывать лишние агрументы в списки или словари:

```py
match user:
    case ("Jack", 42, *args):
        print(f"{args}")
```

- **Проверка типов** — можно проверять, является ли объект экземпляром конкретного класса;

- **Ограничители (Guards)** — в блок `case` можно добавить дополнительное условие `if`:

    ```py
    case [x, y] if x == y:
        print("Координаты равны")
    ```

**Особенности:**

- **Декларативность** — вы описываете, как должны выглядеть данные, а не пишете цепочку проверок `isinstance()` или `len()`;
- **Эффективность** — в большинстве случаев конечно рекомендуется использовать условные операторы `if-elif-else`. Использование `match-case` же идеально подходит для работы со сложными вложенными структурами данных, так как делает код значительно чище и читаемее.

# Циклы

**Цикл в программировании** — это [управляющая конструкция](glossary.md#управляющая-конструкция), которая позволяет многократно выполнять один и тот же участок кода — *тело цикла*, пока **выполняется заданное условие**.

**Тело цикла** — это [блок кода](glossary.md#блок-кода), выполняющийся каждую **итерацию цикла**.

Одно **выполнение тела цикла**, называется **итерацией**.

## Цикл `for`

**Цикл `for`** — последовательно перебирает элементы заданной коллекции или итерируемого объекта, выполняя тело цикла для каждого из них.

```py
for i in range(3):
    print('Hello World!') # тело цикла

Hello World!
Hello World!
Hello World!
```

**Примечания:**

- Если переменная цикла `i` **не используется** в теле цикла, вместо неё **указывается символ нижнего подчеркивания `_`**.
- Название переменной не всегда указывается как `i` и может быть любым **допустимым** названием переменной.

---

## Цикл while

**Цикл `while`** — **проверяет заданное условие** и выполняет **тело цикла**, пока **условие истинно**.

```py
i = 3
while i > 0:
    print('Hello world!')
    i -= 1

Hello World!
Hello World!
Hello World!
```

**Примечания:**

- Условие в цикле `while`, как и в условном операторе `if`, может содержать логические операторы `or`, `and`, `not`.
- Цикл `while` является **бесконечным**, если его условие **всегда истинно**.
- Если **условие ложно**, цикл не выполнит **ни одной итерации**.

---

## Управление циклом: break, continue, else

**Оператор `break`** - позволяет **прервать** выполнения цикла **преждевременно**.

```py
i = 3
while i > 0:
    if i == 2:
        break # выполнение прервано
    print('Hello world!')
    i -= 1

Hello World!
```

---

**Оператор `continue`** - позволяет **пропустить текущую итерацию цикла** и перейти к следующей.

```py
for i in range(3):
    if i == 1:
        continue
    print(i)
0
2
```

---

**Блок `else`** - python допускает **необязательный блок else** в конструкции цикла. Этот блок выполняется только в том случае, если цикл **завершлися без прерывания оператором break.**

```py
a, b = int(input()), int(input())

for i in range(a, b) :
    if i == 2:
        print('Exit')
        break
else:
    print('Done')
```

В примере выше, если числовой диапазон `[a, b)` содержит число `2` цикл завершается с выводом `Exit` в консоль и блок `else` не выполняется. Если же диапазон не содержит число `2`, блок `else` выполнится и в выводе будет `Done`.
