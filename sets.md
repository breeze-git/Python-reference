# Что такое множество?

**Множество (set)** — это **изменяемая неупорядоченная [коллекция](glossary.md#коллекция) уникальных** элементов **[хешируемых](glossary.md#хешируемый-объект)** типов.

**Синтаксис создания множества:**

```py
st = {1, 'two', (3, )}
print(st)

{1, 'two', (3,)}
```

Множества можно создавать при помощи **встроенной функции `set()`**, которая помимо создания множества может **преобразовывать другие итерируемые объекты в множества**.

Например мы можем создать множество из списка с помощью функции `set()`:

```py
lst = [1, 3, 2, 4, 1, 3, 2]

st = set(lst)

print(st)

{1, 2, 3, 4}
```

Как вы можете видеть в примере выше `set()` принимает список и **возвращает множество**, содержащее все его **уникальные элементы, дубликаты автоматически отбрасываются**.

**Примечания:**

- Множество, как и список или кортеж, может содержать значения **разных типов данных**.
- Создать **пустое множество** можно только функцией `set()`.
- Функция `print()` выводит на экран элементы множества, **в фигурных скобках, разделенные запятыми**.
- Множество **не может содержать списки, множества или другие изменяемые структуры данных**, так как они **не являются хешируемыми**. Попытки создать множество с нехешируемыми элементами, **вызывают ошибку `TypeError`**.
- Индексация и срезы **недоступны для множеств**.
- Операции конкатенации `+` и умножение на число `*` **недоступны для множеств**.
- Множества поддерживает быстрые операции проверки наличия, добавления и удаления элементов, операции объединения, пересечения и многие другие.

# Оператор `in` и итерирование множеств

**Оператор принадлежности `in`** — позволяет проверить, **находится ли указанный элемент в составе множества**.

```py
st = {'one', 'two', 'three'}
print('one' in st)

True
```

Как и вслучае с другими итерируемыми объектами, с помощью цикла `for` и оператора `in` мы можем **итерировать и множества**.

```py
st = {'one', 'two', 'three'}
for e in st:
    print(e, end = ' ')

one two three
```

# Работа с множествами. Встроенные функции `len()`, `sum()`, `min()`, `max()`

**Функция `len()`** — возвращает длину множества.

```py
st = {0, 1, 2}

print(len(st))

3
```

---

**Функция `sum()`** — принимает в качестве параметра множество чисел и вычисляет сумму его элементов.

```py
print(sum({1, 3, 4}))

8
```

---

**Функция `min()`** — принимает в качестве параметра множество и возвращает **минимальный элемент** этого множества.

```py
print(min({1, 3, 7}))

1
```

---

**Функция `max()`** — принимает в качестве параметра множество и возвращает **максимальный элемент** этого множества.

```py
print(max({1, 7, 10}))

10
```

**Примечания:**

- Функция `sum()` принимает **только числовые аргументы**. Если в качестве аргумента задать **список строк, это вызовет ошибку.**
- Функции `min()` и `max()` **работают как с числами, так и со строками.** Но все **аргументы в списке должны быть одного типа, иначе это вызовет ошибку сравнения**.

# Вывод элементов множества

**Есть несколько способов вывести содержимое множества.**

**`print(st)`** — простой вывод множества через функцию `print()`. При таком выводе все элементы множества `st` выводятся в **фигурных скобках** и разделены запятыми.

```py
st = {1, 2, 3}
print(st)

{1, 2, 3}
```

---

**`print(*st)`** — вывод с помощью [распаковки](operators.md#операции-распаковки-и-упаковки-итерируемых-объектов-оператор--в-распаковке-и-упаковке) множества, все элементы выводятся **через один символ пробела**.

```py
st = {1, 2, 3}
print(*st)

1 2 3
```

Также мы можем вывести каждый элемент **на отдельной строке**, если добавим параметр `sep` функции `print()` со значением `'\n'`

```py
st = {1, 2, 3}
print(*st, sep = '\n')

1
2
3
```

Пустое множество выводится как `set()`, а не как `{}`. С помощью `{}` выводится пустой словарь.

```py
st = set()

print(st)

set()
```

**Примечания:**

- Множества – неупорядоченные коллекции, поэтому полагаться на порядок вывода элементов не стоит.

# Генераторы множеств

**Синтаксис:**

`{выражение for переменная in последовательность if условие}`

- где `выражение` – некоторое выражение, как правило, зависящее от использованной в генераторе множеств переменной, которым будут заполнены элементы множества, `переменная` – имя некоторой переменной, `последовательность` – последовательность значений, которые она принимает (любой итерируемый объект), `условие` (необязательное) – в множество попадут элементы, для которых условие истинно.

К примеру так мы можем получить множество всех чисел диапазона `[125, 298)`, которые без остатка делятся на число `73`:

```py
nums = {i for i in range(125, 298) if i % 73 == 0 }

print(nums)

{146, 219, 292}
```

# Сравнение множеств

Для сравнения множеств используются операторы `==`, `!=`, `>=`, `<=`, `>`, `<`.

Одно множество, является подмножеством другого множества, если все элементы первого входят во второе, и в этом случае второе множество является надмножеством первого.

Например множество `{1, 2, 3}` является **подмножеством** множества `{1, 2, 3, 4, 5}`. И множество `{1, 2, 3, 4, 5}` является **надмножеством** множества `{1, 2, 3}`.

---

Два множества **равны**, если они содержат **одинаковые** элементы.

```py
print({1, 2, 3} == {1, 2, 3})

True
```

В примере выше, множество `{1, 2, 3}` также является подмножеством самого себя. Это называется **нестрогое подмножество**.

---

Соответственно множества **не равны**, если они содержат **разные** элементы.

```py
print({1, 2, 3} != {'one', 'two', 'three'})

True
```

---

Для определения, является ли одно из множеств **подмножеством** другого, применяются операторы `<=` (нестрогое подмножество) и `<` (строгое подмножество).

```py
print({1, 2, 3} < {1, 2, 3, 4, 5}, 
      {1, 2, 3} <= {1, 2, 3})

True
True
```

---

Для определения, является ли одно из множеств **надмножеством** другого, применяются операторы >= (нестрогое надмножество) и > (строгое надмножество).

```py
print({1, 2, 3, 4, 5} > {1, 2, 3}, 
      {1, 2, 3} >= {1, 2, 3})

True
True
```

**Примечания:**

- Для сравнения множеств можно также использовать методы `issuperset()`, `issubset()`, `isdisjoint()`. Подробнее о них [здесь](sets.md#set-operations-methods).
- Операторы `>`, `<`, `>=`, `<=` требуют наличия в качестве операндов **множества**, в то время как методы принимают в качестве **аргумента** любой **итерируемый** объект.

# Операции над множествами

**Объединение множеств** — это множество, состоящее из элементов, принадлежащих хотя бы одному из объединяемых множеств.

Для объединения множеств используется оператор `|`:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st3 = st1 | st2

print(st3)

{1, 2, 3, 4, 5, 6, 7}
```

С помощью оператора `|=` можно изменять исходное множество по объединению:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st1 |= st2

print(st1)

{1, 2, 3, 4, 5, 6, 7}
```

---

**Пересечение множеств** — это множество, состоящее из элементов, принадлежащих одновременно каждому из пересекающихся множеств.

Для пересечения двух множеств используется оператор `&`:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st3 = st1 & st2

print(st3)

{3, 4}
```

С помощью оператора `&=` можно изменять исходное множество по пересечению:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st1 &= st2

print(st1)

{3, 4}
```

---

**Разность множеств** – это множество, в которое входят все элементы первого множества, не входящие во второе множество.

Для разности двух множеств используется оператор `-`:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st3 = st1 - st2

print(st3)

{1, 2}
```

С помощью оператора `-=` можно изменять исходное множество по разности:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st1 -= st2

print(st1)

{1, 2}
```

---

**Симметрическая разность множеств** – это множество, включающее все элементы исходных множеств, не принадлежащие одновременно обоим исходным множествам.

Для симметрической разности двух множеств используется оператор `^`:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st3 = st1 ^ st2

print(st3)

{1, 2, 5, 6, 7}
```

С помощью оператора `^=` можно изменять исходное множество по разности:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st1 ^= st2

print(st1)

{1, 2, 5, 6, 7}
```

---

Приоритет операторов в порядке убывания (верхние операторы имеют более высокий приоритет, чем нижние):

| Оператор | Описание |
|----------|-------------------------|
| `-` | разность |
| `&` | пересечение |
| `^` | симметрическая разность |
| `\|` | объединение |

**Примечания:**

- Для операций с множествами можно также использовать методы `union()`, `intersection()`, `difference()`, `symmetric_difference()`. Подробнее о них [здесь](#методы-операций-с-множествами).
- Операторы `|`, `&`, `-`,`^` требуют наличия в качестве операндов **множества**, в то время как методы принимают в качестве **аргумента** любой **итерируемый** объект.
- Методы `union()`, `intersection()`, `difference()` и операторы `|`, `&`, `-`, `^` позволяют совершать операции над **несколькими** множествами сразу. Метод `symmetric_difference()` – нет.

# Множества `frozenset`

**Замороженное множество (frozenset)** — **неизменяемая** версия обычного множества (set).

Множества frozenset обладают всеми характеристиками обычных множеств, но **не могут быть изменены**.

Для создания замороженного множества используется встроенная функция `frozenset()`, которая принимает в качестве аргумента другую коллекцию.

Например:

```py
frst = frozenset([1, 2, 3])
print(frst)

frozenset({1, 2, 3}) # вывод
```

Как вы могли заметить результатом простого вывода множества `frozenset` через функцию `print()`, является запись с явным указанием на множество `frozenset`.

---

Над замороженными множествами можно производить все операции, которые можно производить над обычными множествами:

- **объединение множеств**: метод `union()` или оператор `|`;
- **пересечение множеств**: метод `intersection()` или оператор `&`;
- **разность множеств**: метод `difference()` или оператор `-`;
- **симметрическая разность множеств**: метод `symmetric_difference()` или оператор `^`.

**Результатом операций над замороженными множествами будут тоже замороженные множества.**

И так как замороженные множества **неизменяемы**, у них **отсутствуют такие методы как**: `add()`, `remove()`, `discard()`, `pop()`, `clear()`, `update()`, `intersection_update()`, `difference_update()`, `symmetric_difference_update()`. Также операторы : `|=`, `&=`, `-=`, `^=` **не доступны для них**.

**Примечания:**

- Замороженные множества являются неизменяемыми, а значит могут быть элементами других множеств.
- Мы можем сравнивать простые (set) и замороженные множества (frozenset).

# Методы множеств

**Методы** — это набор функций, принадлежащий объектам определенного класса.

Подробнее что такое методы читайте [здесь](glossary.md#метод).

## Добавление элементов

**`st.add()`** — позволяет добавить новый элемент в множество.

```py
st = {'a', 'b', 'c'}

st.add('d')

print(st)

{'a', 'd', 'b', 'c'}
```

## Удаление элементов

**`st.remove(elem)`** — позволяет **удалить элемент `elem`** из множества. В случае, если такого элемента нет **вызывает ошибку** `KeyError`.

```py
st = {'Will', 'Turner', 23}

st.remove(23)

print(st)

{'Turner', 'Will'}
```

---

**`st.discard(elem)`** — позволяет **удалить элемент `elem`** из множества, **без генерации ошибки**, если элемент отсутствует в нем.

```py
st = {'Will', 'Turner', 23}

st.discard('Jack') # не вызывает ошибку
st.discard(23)

print(st)

{'Turner', 'Will'}
```

---

**`st.pop()`** — позволяет **удалить случайный элемент** из множества и возвращает удаленный элемент. При попытке удаления из пустого множества **вызывает ошибку** `KeyError`.

```py
st = {'Will', 'Turner', 23}

st.pop()

print(st)

{'Turner', 23}
```

---

**`st.clear()`** — позволяет **удалить все элементы** из множества.

```py
st = {'Will', 'Turner', 23}

st.clear()

print(st)

set() # выводит пустое множество
```

## Методы сравнения множеств

**`st1.issubset(st2)`** — возвращает значение `True`, если множество `st1` является подмножеством `st2`, в противном случае возвращает `False`.

Параметр `st2` может быть не только множеством, а любым итерируемым объектом.

```py
st1 = {2, 3, 4}
st2 = {1, 2, 3, 4, 5}

print(st1.issubset(st2))

True
```

---

**`st1.issuperset(st2)`** — возвращает значение `True`, если множество `st1` является надмножеством `st2`, в противном случае возвращает `False`.

Параметр `st2` может быть не только множеством, а любым итерируемым объектом.

```py
st1 = {1, 2, 3, 4, 5}
st2 = {2, 3, 4} 

print(st1.issuperset(st2))

True
```

---

**`st1.isdisjoint(st2)`** — возвращает значение `True`, если множества `st1` и `st2` не имеют общих элементов, и `False`, в противном случае.

Параметр `st2` может быть не только множеством, а любым итерируемым объектом.

```py
st1 = {1, 2, 3, 4, 5}
st2 = {2, 3, 4} 

print(st1.isdisjoint(st2))

False
```

## Методы операций с множествами

**`st1.union(st2)`** — возвращает новое множество в которое входят **все элементы множеств `st1` и `st2`**(**объединение** множеств).

Параметр `st2` может быть не только множеством, а любым итерируемым объектом.

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st3 = st1.union(st2)

print(st3)

{1, 2, 3, 4, 5, 6, 7}
```

Для изменения **текущего** множества используется метод **`st1.update(st2)`**:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st1.update(st2)

print(st1)

{1, 2, 3, 4, 5, 6, 7}
```

---

**`st1.intersection(st2)`** — возвращает новое множество в которое входят о**бщие элементы множеств `st1` и `st2`**(**пересечение** множеств).

Параметр `st2` может быть не только множеством, а любым итерируемым объектом.

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st3 = st1.intersection(st2)

print(st3)

{3, 4}
```

Для изменения **текущего** множества используется метод **`st1.intersection_update(st2)`**:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st1.intersection_update(st2)

print(st1)

{3, 4}
```

---

**`st1.difference(st2)`** — возвращает новое множество в которое входят элементы, **входящие в множество `st1` и которых нет в множестве `st2`**(**разность** множеств).

Параметр `st2` может быть не только множеством, а любым итерируемым объектом.

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st3 = st1.difference(st2)

print(st3)

{1, 2}
```

Для изменения **текущего** множества используется метод **`st1.difference_update(st2)`**:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st1.difference_update(st2)

print(st1)

{1, 2}
```

---

**`st1.symmetric_difference(st2)`** — возвращает новое множество в которое входят элементы исходных множеств, **не принадлежащие одновременно обоим исходным множествам `st1` и `st2`**(**симметрическая разность** множеств).

Параметр `st2` может быть не только множеством, а любым итерируемым объектом.

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st3 = st1.symmetric_difference(st2)

print(st3)

{1, 2, 5, 6, 7}
```

Для изменения **текущего** множества используется метод **`st1.symmetric_difference_update(st2)`**:

```py
st1 = {1, 2, 3, 4}
st2 = {3, 4, 5, 6, 7}

st1.symmetric_difference_update(st2)

print(st1)

{1, 2, 5, 6, 7}
```
