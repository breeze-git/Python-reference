# Что такое кортеж?

**Кортеж (tuple)** — это **неизменяемая упорядоченная последовательность элементов произвольных типов**, индексируемых целыми числами.

**Синтаксис создания кортежа:**

```py
lst = (1, 2, 3)
print(l)

(1, 2, 3)
```

Кортежи можно создавать при помощи **встроенной функции `tuple()`**, которая помимо создания кортежа может **преобразовывать другие итерируемые объекты в кортежи**.

Например мы можем создать кортеж при помощи функции `list()` и встроенной [функции `range()`](builtins.md#часто-используемые-функции):

```py
tpl = tuple(range(1,5))
print(tpl)

(1, 2, 3, 4)
```

Также мы можем создать **кортеж из строки**:

```py
s = 'Hello World!'
tpl = tuple(s)

print(tpl)

('H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd', '!')
```

Или из **списка**:

```py
lst = [1, 2, 3]
tpl = tuple(lst)

print(tpl)

(1, 2, 3)
```

**Примечания:**

- Кортеж, как и список, может содержать значения **разных типов данных**.
- Создать **пустой кортеж** можно двумя способами: **пустыми скобками `()` и функцией `tuple()`**.
- Чтобы создать кортеж с единственным элементом после значения элемента ставят замыкающую запятую: `tpl = (Will,)`.
- Функция `print()` выводит на экран элементы кортежа, **в круглых скобках, разделенные запятыми**.
- Обработка кортежа выполняется **быстрее, чем обработка списка**, так как из-за неизменяемости их элементы хранятся в памяти иначе и **операции с ними выполняются заведомо быстрее**.
- В кортеже можно **безопасно хранить данные**, не опасаясь, что они будут **случайно или преднамеренно изменены** в программе, так как **неизменяемость** превращает кортежи в **идеальные константы**.

# Индексация кортежей. Срезы

[Индексация](glossary.md#индексация-элементов) кортежей в Python, работает так же, как и в строках или списках. Подробнее об этом вы можете прочитать [здесь](strings.md#индексация-строк-срезы).

```py
tpl = (1, 2, 3, 4, 5)
print(tpl[0], tpl[-1])

1 5
```

С помощью среза мы можем **получить несколько элементов кортежа**, создав диапазон индексов, разделенных двоеточием numbers[x:y].

**Срезы в котежах работают аналогично срезам в строках**. Подробнее о них вы можете прочитать так же [здесь](strings.md#индексация-строк-срезы).

**Примечания:**

- Кортежи неизменяемы, поэтому доступ к элементам по индексу возможен только для получения значений элементов.

# Оператор `in` и итерирование кортежей

**Оператор принадлежности `in`** — позволяет проверить, **находится ли указанный элемент в составе кортежа**.

```py
tpl = ('one', 'two', 'three')
print('one' in tpl)

True
```

**Кортежи — итерируемые объекты**. Мы можем перебирать их элементы с помощью цикла `for` и оператора `in`.

```py
lst = ('one', 'two', 'three')
for e in lst:
    print(e, end = ' ')

one two three
```

Таже мы можем перебирать элементы по индексам, используя [функцию `len()`](builtins.md#часто-используемые-функции), чтобы узнать длину кортежа.

```py
tpl = ('one', 'two', 'three')
for i in range(len(tpl)):
    print(tpl[i], end = ' ')

one two three
```

# Конкатенация и умножение кортежей на число

**Конкатенация** — операция, позволяющая **складывать кортежи** с помощью оператора сложения — `+`.

```py
print((1, 2, 3) + (4, ))

(1, 2, 3, 4)
```

---

**Операция умножения кортежа на число** - позволяет умножить кортеж на целое неотрицательное число, при помощи оператора умножения — `*`, который повторяет кортеж указанное количество раз.

```py
print((1, 3) * 3)

(1, 3, 1, 3, 1, 3)
```

# Сравнение кортежей

**Сравнение кортежей идентично сравнению строк:**

1. Сравниваются **первые элементы** кортежей. Если элементы равны, переход к следующей паре элементов.
2. Когда находится первый отличающийся элемент, **кортеж с меньшим элементом считается `меньше`**.
3. Если один из кортежей **заканчивается раньше**  другого, то он считается  **`меньшим`**.

**Пример сравнения кортежей:**

```py
print((1, 2 , 3) < (2, 10, 17),
      (1, 2 , 3) < (1, 2 , 3, 1),
)

True True
```

Операции `==` и `!=` применимы к любым кортежам, **независимо от типов элементов**.

```py
print((1, 2, 3) != ('one', 'two', 'three'),
      (1, ) == ('one', )
)

True False
```

Операции `<`, `>`, `<=`, `>=` применимы только в том случае, когда соответствующие элементы кортежей имеют **один тип**.

```py
print(('Will', 'male', 21) > ('Bill', 'male', 53))

True

print(('Will', 'male', 21) > (53, 'male', 'Bill'))

TypeError: '>' not supported between instances of 'str' and 'int' # ошибка
```

# Работа с кортежами. Встроенные функции `sum()`, `min()`, `max()`

**Функция `sum()`** —  принимает в качестве параметра кортеж чисел и вычисляет сумму его элементов.

```py
print(sum((1, 3, 4)))

8
```

---

**Функция `min()`** — принимает в качестве параметра кортеж и возвращает **минимальный элемент** этого кортежа.

```py
print(min((1, 3, 7)))

1
```

---

**Функция `max()`** — принимает в качестве параметра кортеж и возвращает **максимальный элемент** этого кортежа.

```py
print(max((1, 7, 10)))

10
```

Функции `min()` и `max()` могут работать и с **кортежами**. Если этим функциям передается **несколько кортежей, сравнение происходит поэлементно(как со строками и списками).**

```py
print(min((1, 2, 3), (3, 2, 1)))

(1, 2, 3)

print(max(('one', ), ('two', )))

('two',)
```

**Примечания:**

- Функция `sum()` принимает **только числовые аргументы**. Если в качестве аргумента задать **кортеж строк, это вызовет ошибку.**
- Функции `min()` и `max()` **работают как с числами, так и со строками.** Но все **аргументы в кортеже должны быть одного типа, иначе это вызовет ошибку сравнения**.

# Упаковка и распаковка кортежа

**Упаковка** — это операция, при которой несколько значений, перечисленных через запятую, **автоматически объединяются в кортеж**.

**Распаковкой кортежа** — присвоение значений элементов кортежа отдельным переменным.

Подробнее об этих операциях [здесь](operators.md#операции-распаковки-и-упаковки-итерируемых-объектов-оператор--в-распаковке-и-упаковке).

# Вывод элементов кортежа

**Есть несколько способов вывести содержимое кортежа.**

**`print(tpl)`** — простой вывод кортежа через функцию `print()`. При таком выводе все элементы кортежа `tpl` выводятся в **круглых скобках** и разделены запятыми.

```py
tpl = (1, 2, 3)
print(tpl)

(1, 2, 3)
```

---

**`print(*tpl)`** — вывод с помощью [распаковки](operators.md#операции-распаковки-и-упаковки-итерируемых-объектов-оператор--в-распаковке-и-упаковке) кортежа, все элементы выводятся **через один символ пробела**.

```py
tpl = (1, 2, 3)
print(*tpl)

1 2 3

```

Также мы можем вывести каждый элемент **на отдельной строке**, если добавим параметр `sep` функции `print()` со значением `'\n'`

```py
tpl = (1, 2, 3)
print(*tpl, sep = '\n')

1
2
3
```

---

# Вложенные кортежи

Элементами кортежей могут быть другие кортежи. Такие кортежи называются **вложенными кортежами**.

Чтобы обратиться к элементу вложенного кортежа, нужно указать индекс внешнего кортежа, а затем индекс во внутреннем кортеже.

```py
lst = (1, 2, (10, 20))

print(lst[2][0])

10
```

# Методы кортежей

**Методы** — это набор функций, принадлежащий объектам определенного класса.

Подробнее что такое методы читайте [здесь](glossary.md#метод).

## Поиск

**`t.index(value)`** — **возвращает индекс первого элемента** в кортеже, значение которого **равняется** `value`.
Если элемент в кортеже **не найден**, то во время выполнения происходит **ошибка**.

```py
t = (1, 2, 3)

print(t.index(3))

2
```

---

**`t.count(value)`** — возвращает количество элементов в кортеже, значения которых **равны** `value`. Если значение в кортеже **не найдено**, то метод **возвращает `0`**.

```py
t = (1, 2, 3)

print(t.count(3))

1
```
