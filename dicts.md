# Что такое словарь?

**Словарь (dict)** — это **изменяемая [коллекция](glossary.md#коллекция), в которой элементы хранятся как пары «ключ: значение»**, и доступ к значению элемента осуществляется по его **ключу**.

**Синтаксис создания словаря:**

```py
dct = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}
print(dct)

{'name': 'Jack', 'surname': 'Sparrow', 'age': 34}
```

---

**Ключ** — уникальный идентификатор, по которому мы получаем доступ к значению элемента в словаре.

```py
dct = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

print(dct['name'], dct['surname'])

Jack Sparrow
```

Все ключи в словаре должны быть уникальными. Также ключами могут быть только [хешируемые](glossary.md#хешируемый-объект) типы данных. Если мы попытаемся сделать ключом список элементов, то получим ошибку `TypeError`.

```py
dct = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34, [1, 2, 3]: 123}

print(dct)

TypeError: unhashable type: 'list' # ошибка
```

При попытке извлечь значение по несуществующему ключу возникнет ошибка `KeyError`:

```py
dct = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

print(dct['job'])

KeyError: 'job' # ошибка
```

---

**Примечания:**

- Функция `print()` выводит на экран элементы словаря, **в фигурных скобках, разделенные запятыми**.
- Если требуется вывести только значение **ключей** словаря, то мы можем использовать операцию [распаковки](operators.md#операции-распаковки-и-упаковки-итерируемых-объектов-оператор--в-распаковке-и-упаковке) словаря.
- Индексация и срезы **недоступны для словарей**.
- Операции конкатенации `+` и умножение на число `*` **недоступны для словарей**.
- В рамках одного словаря каждый ключ **уникален**.
- В словаре используется специальная структура данных — **хеш-таблица**. Она позволяет вычислять **числовой хеш** от ключа и использовать обычные списки, где в качестве индекса элемента берется этот хеш.
- Ключами словаря могут быть только **хешируемые** типы данных

# Встроенная функция `dict()`

**Функция `dict()`** позволяет создать словарь несколькими способами.

**Создание пустого словаря:**

```py
dct = dict()

print(dct)

{}
```

**Создание словаря из пар ключ-значение:**

```py
dct = dict(name = 'Jack', surname = 'Sparrow', age = 34)

print(dct)

{'name': 'Jack', 'surname': 'Sparrow', 'age': 34}
```

Ключи должны быть [валидными именами](basics.md#именование-переменных) переменных.

**Создание словаря из итерируемого объекта пар (например, списка кортежей):**

```py
dct = dict([('name', 'Jack'), ('surname', 'Sparrow'), ('age', 34)])

print(dct)

{'name': 'Jack', 'surname': 'Sparrow', 'age': 34}
```

Подходит любая коллекция, где каждый элемент — последовательность из двух элементов (список кортежей, кортеж списков, генератор пар и т.д)

**Копирование словаря:**

```py
dct = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}
print(dct)

dct_copy = dict(dct)

print(dct_copy)

{'name': 'Jack', 'surname': 'Sparrow', 'age': 34}
```

Создает [поверхностую копию](glossary.md#поверхностная-копия) словаря.

# Изменение и удаление элементов

Мы можем изменить значение элемента в словаре, обращаясь к нему по его ключу:

```py
dct = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

dct['age'] = 35

print(dct)

{'name': 'Jack', 'surname': 'Sparrow', 'age': 35}
```

---

**Оператор `del`** — удаляет из словаря элемент **по заданному ключу** вместе с его значением. В случае отсутствия элемента **вызывает ошибку** `KeyError`.

```py
dct = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

del dct['age']

print(dct)

{'name': 'Jack', 'surname': 'Sparrow'}
```

# Оператор `in` и итерирование словарей

**Оператор принадлежности `in`** — позволяет проверить, **содержит ли словарь заданный ключ**.

```py
dct = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

print('age' in dct)

True
```

С помощью цикла `for` и оператора `in` мы можем **итерировать ключи словаря**.

```py
dct = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

for key in dct:
    print(dct[key], end = ' ')

Jack Sparrow 34
```

# Работа со словарями. Встроенные функции `len()`, `sum()`, `min()`, `max()`

**Функция `len()`** — возвращает длину словаря.

```py
dct = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

print(len(dct))

3
```

---

**Функция `sum()`** —  принимает в качестве параметра  словарь с числовыми ключами и **вычисляет сумму его ключей**.

```py
dct = {1: 'one', 2: 'two', 3: 'three'}

print(sum(dct))

6
```

---

**Функция `min()`** — принимает в качестве параметра словарь и возвращает **минимальный ключ** этого словаря, при этом ключ может принадлежать к любому типу данных, для которого возможны операции сравнения (числа, строки, и т.д.).

```py
dct = {1: 'one', 2: 'two', 3: 'three'}

print(min(dct))

1
```

---

**Функция `max()`** — принимает в качестве параметра словарь и возвращает **максимальный ключ** этого словаря, при этом ключ может принадлежать к любому типу данных, для которого возможны операции сравнения (числа, строки, и т.д.).

```py
dct = {1: 'one', 2: 'two', 3: 'three'}

print(max(dct))

3
```

# Сравнение словарей

Для сравнения словарей используются операторы `==` и `!=`.

Равные словари имеют одинаковое количество элементов и содержат равные элементы (ключ: значение).

```py
dct1 = {1: 'one', 2: 'two', 3: 'three'}
dct2 = dict(dct1)

print(dct1 == dct2)

True
```

# Вложенные словари

Значениями элементов словарей могут быть другие словари. Такие словари называются **вложенными словарями**.

Чтобы обратиться к элементу вложенного словаря, нужно указать ключ элемента внешнего словаря, а затем ключ элемента во внутреннем словаре.

```py
dct = {'name': 'Bill', 'surname': 'Turner', 'age': 53, 'son': {'name': 'Will', 'surname': 'Turner', 'age': 24}}

print(dct['son']['name'], dct['son']['age'])

Will 24
```

# Генераторы словарей

**Синтаксис:**

`{ключ: значение for переменная in последовательность if условие}`

где `переменная` — имя некоторой переменной, `последовательность` — последовательность значений, которые она принимает (любой итерируемый объект), `ключ: значение` — некоторое выражение, как правило, зависящее от использованной в генераторе словаря переменной, которой будут заполнены элементы словаря, `условие` (необязательное) – в словарь попадут элементы, для которых условие истинно.

К примеру мы можем заполнить словарь квадратами чисел некоторого диапазона, используя само число как ключ, а значение элемента - квадрат этого числа:

```py
dct = {num: num**2 for num in range(10, 15)}

print(dct)

{10: 100, 11: 121, 12: 144, 13: 169, 14: 196}
```

# Методы словарей

Что такое методы читайте [здесь](glossary.md#метод).

## Получение элементов

**`d.keys()`** — возвращает список **ключей** всех элементов словаря.

```py
d = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

print(d.keys())

dict_keys(['name', 'surname', 'age']) # вывод
```

Обратите внимание, что метод возвращает не список `list`, а объект типа `dict_keys`. Чтобы применить к нему методы обычных списков необходимо преобразовать его с помощью функции `list()`.

---

**`d.values()`** — возвращает список **значений** всех элементов словаря.

```py
d = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

print(d.values())

dict_values(['Jack', 'Sparrow', 34])
```

Обратите внимание, что метод возвращает не список `list`, а объект типа `dict_values`. Чтобы применить к нему методы обычных списков необходимо преобразовать его с помощью функции `list()`.

---

**`d.items()`** — возвращает список **всех элементов** словаря, состоящий из **кортежей пар** (ключ, значение).

```py
d = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

print(d.items())

dict_items([('name', 'Jack'), ('surname', 'Sparrow'), ('age', 34)])
```

Обратите внимание, что метод возвращает не список `list`, а объект типа `dict_items`. Чтобы применить к нему методы обычных списков необходимо преобразовать его с помощью функции `list()`.

## Добавление и изменение элементов

**`d.get(key, defaultvalue=None)`** — позволяет **получить значение** элемента в словаре **по ключу** `key`. В **случае отсутствия ключа** в словаре метод **возвращает значение необязательного параметра `defaultvalue`**, по умолчанию равное `None`.

```py
d = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

print(d.get('job', 'Pirate'))

Pirate
```

---

**`d1.update(d2)`** — объединяет ключи и значения словаря `d1` с ключами и значениями словаря `d2`. При совпадении ключей в итоге сохранится значение словаря `d2`, указанного в качестве аргумента.

```py
d1 = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34, 'job': 'Pirate'}

d2 = {'name': 'Jack', 'surname': 'Sparrow', 'age': 35, 'job': 'Captain'}

d1.update(d2)

print(d1)

{'name': 'Jack', 'surname': 'Sparrow', 'age': 35, 'job': 'Captain'}
```

**Примечания:**

- Метод в качестве аргумента может также принимать последовательность пар элементов(список кортежей, кортеж списков и т.д).
- В Python 3.9 появились операторы `|` и `|=`, позволяющие объединять словари.

---

**`d.setdefault(key, default=None)`** — позволяет получить значение из словаря по **заданному ключу** `key`, автоматически **добавляя элемент словаря, если он отсутствует**.

```py
d = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

print(d.setdefault('job', 'Pirate'), d, sep = '\n')

Pirate
{'name': 'Jack', 'surname': 'Sparrow', 'age': 34, 'job': 'Pirate'}

```

Если ключ `key` **присутствует в словаре**, то метод возвращает значение по заданному ключу (независимо от того, передан параметр `default` или нет).

---

**`dict.fromkeys(keys, value=None)`** — создает словарь с заданными ключами `keys` и одинаковыми значениями `value`, по умолчанию равному `None`.

Параметр `keys` принимает **итерируемый объект** (список, кортеж, строка и т.д.), содержащий ключи будущего словаря.

```py
d = dict.fromkeys([1, 2, 3], 'number')

print(d)

{1: 'number', 2: 'number', 3: 'number'}
```

---

**`d.copy()`** — создает [поверхностную копию](glossary.md#поверхностная-копия) словаря.

```py
d1 = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}
d2 = d1.copy()

print(d2)

{'name': 'Jack', 'surname': 'Sparrow', 'age': 34}
```

## Удаление элементов

**`d.pop(key, default=None)`** — позволяет удалить элемент словаря по ключу `key` и возвращает его значение. В случае отсутствия элемента вызывает ошибку `KeyError`. Методу можно передать второй аргумент `default`, который будет возвращен вместо вызова ошибки, в случае отсутствия элемента в словаре.

```py
d = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

print(d.pop('age'))

34
```

---

**`d.popitem()`** — удаляет из словаря **последний** добавленный элемент и **возвращает** удаляемый элемент в виде **кортежа** (ключ, значение).

```py
d = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

print(d.popitem(), d, sep = '\n')

('age', 34)
{'name': 'Jack', 'surname': 'Sparrow'}
```

**Примечания:**

- В версиях Python ниже 3.6 метод `popitem()` удалял случайный элемент.

---

**`d.clear()`** —  удаляет все элементы из словаря.

```py
d = {'name': 'Jack', 'surname': 'Sparrow', 'age': 34}

d.clear()

print(d)

{}
```
