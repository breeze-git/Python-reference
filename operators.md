# Арифметические операторы

**Базовые арифметические операторы:**

`+` — бинарный плюс

______________________________________________________________________

`-` — бинарный минус

______________________________________________________________________

`*` — умножение

______________________________________________________________________

`/` — деление

Результат **операции /**, всегда является число с **плавающей точкой**, даже если оба операнда являются **целыми числами**.

______________________________________________________________________

**Помимо базовых арифметических операторов в Python есть также дополнительные операторы:**

`**` — возведение в степень

Возведение в степень является правоассоциативным (значение выражения вычисляется справа налево) в соответствии с правилами математики. Таким образом, выражение `x ** y ** z` вычисляется как `x ** (y ** z)`.

С помощью выражения `n ** 0.5` можно получить квадратный корень числа `n`.

______________________________________________________________________

`//` — целочисленное деление

При делении отрицательных чисел необходимо помнить, что округление всегда берётся в меньшую сторону.

```py
print(-10//4)
-3
```

Если один из операндов принадлежит типу `float`, результатом операции будет значение типа `float`.

______________________________________________________________________

`%` — деление с остатком

______________________________________________________________________

`-` — унарный минус

# Операторы сравнения

**Операторы сравнения** — это выражения, которые проверяют отношения между значениями и **возвращают логическое значение** `True` или `False`.

Python существует 6 основных операторов сравнения.

| Оператор | Описание |
|-----------|----------------------|
| `>` | **больше** |
| `<` | **меньше** |
| `>=` | **больше либо равно**|
| `<=` | **меньше либо равно**|
| `==` | **равно** |
| `!=` | **неравно** |

Операторы сравнения в Python можно объединять в цепочки, например, `a == b == c` или `1 <= x <= 10`. Если `a == b` и `b == c`, то из этого следует, что `a == c`.

# Логические операторы

В Python есть три **логических оператора**, которые позволяют создавать сложные условия.

**Оператор `and`** — возвращает истину, только если все условия истинны.

**Таблица истинности для оператора and:**

| a | b |a and b |
|-------|-------|---------|
|False |False |**False**|
|False |True |**False**|
|True |False |**False**|
|True |True |**True** |

______________________________________________________________________

**Оператор `or`** — возвращает истину, если хотя бы одно из условий истинно.

**Таблица истинности для оператора or:**

| a | b |a or b |
|-------|-------|---------|
|False |False |**False**|
|False |True |**True** |
|True |False |**True** |
|True |True |**True** |

______________________________________________________________________

**Оператор `not`** - возвращает противоположное значение. Если условие истинно, возвращает `False`, если условие ложно, возвращает `True`.

| a | not a |
|-------|-------|
|True |False |
|False |True |

______________________________________________________________________

# Побитовые операторы

`<<` — сдвиг влево

Сдвигает биты числа влево на заданное количество позиций.

______________________________________________________________________

`>>` — сдвиг вправо

Сдвигает биты числа вправо на заданное количество позиций.

______________________________________________________________________

`&` — побитовое `И`

Побитовая операция `И` над числами: если оба бита равны `1` , то результат равен `1` . В противном случае - `0`.

______________________________________________________________________

`|` — побитовое `ИЛИ`

Побитовая операция `ИЛИ` над числами: если оба бита равны `0` , то результат равен `0` . В противном случае - `1`.

______________________________________________________________________

`^` — побитовое `ИСКЛЮЧАЮЩЕЕ ИЛИ`

Побитовая операция `ИСКЛЮЧАЮЩЕЕ ИЛИ`: если оба бита ( `1` или `0` ) одинаковы, результатом будет `0`. В противном случае - `1`.

______________________________________________________________________

`~`— побитовое `НЕ`

Побитовая операция `НЕ` для числа `x` соответствует `-(x+1)`.

# Операторы присваивания

**Оператор присваивания `=`** — выполняет задачу связывания значения, расположенного справа от него, с именем переменной, указанной слева.

```py
num = 13
print(num)
13
```

______________________________________________________________________

**Множественное присваивание** — это одновременное присваивание значений **нескольким переменным**. Слева от знака `=` через запятую указываются имена переменных, а справа — значения. Переменным присваиваются значения в том порядке, в котором они прописаны.

```py
name, surname = 'Will', 'Turner'
print('Имя:', name, 'Фамилия:', surname)
Имя: Will Фамилия: Turner
```

______________________________________________________________________

# Операции распаковки и упаковки итерируемых объектов. Оператор `*` в распаковке и упаковке

**<b id="#unpacking">Распаковка</b>** — операция, позволяющая присвоить элементы итерируемого объекта отдельным переменным.

Пример распаковки списка:

```py
pirate = ['Will', 'male', 21]

name, gender, age = pirate

print(name, gender, age)

Will male 21
```

Количество переменных должно точно совпадать с количеством элементов, иначе это вызовет **ошибку**.

```py
pirate = ['Will', 'male', 21]

name, gender = pirate

print(name, gender, age)

ValueError: too many values to unpack # ошибка
```

______________________________________________________________________

**Оператор `*`** — оператор распаковки/упаковки элементов.

**Он имеет два основных режима:**

- В левой части присваивания он выполняет функцию сборщика элементов: собирает «остаточные» элементы в список.
- В правой части присваивания или аргументах он раскрывает итерируемый объект на отдельные элементы.

Например его можно использовать **при распаковке** итерируемого объекта **в переменные**.

```py
pirate = ['Will', 'male', 21, False]

name, *other, gun = pirate

print(name, gun)
print(other)

Will False
['male', 21]
```

Мы использовали оператор `*` в создании списка `*other` и **собрали в него все "остаточные" элементы присваивания**, при этом нужные нам элементы получили **отдельные переменные** `name` и `gun`.

**Примечание:**

- Переменная, объявленная с оператором `*` всегда создается списком, даже если в него попадает всего один элемент или ни одного.

Так же мы можем использовать оператор `*` **в выводе содержимого объекта в консоль**:

```py
print(*[10, 20, 30])

10 20 30
```

Или **в распаковке внутри списка**:

```py
lst = [0, *[1, 2], 3]
print(lst)

[0, 1, 2, 3]
```

______________________________________________________________________

**Упаковка** — это операция, при которой несколько значений, перечисленных через запятую, **автоматически объединяются в** <a href="tuples.md#tuple">кортеж</a>.

```py
pirate = 'Jack', 'male', '35', True

print(pirate)

('Jack', 'male', '35', True)
```

Упаковка происходит даже с **одним значением**, если после него есть **запятая**.

```py
pirate = 'Jack', 

print(pirate)

(Jack,)
```

# Операторы инструкции

**Оператор `del`** — это встроенная инструкция языка Python, предназначенная для **удаления привязок имён к объектам или удаления элементов из изменяемых контейнеров** (списков, словарей).

```py
l = [1, 2, 3, 4]
del l[3]

print(l)

[1, 2, 3]
```
