# Глоссарий

## Динамическая типизация

**Python** — язык с **динамической типизацией**. Это означает, что переменная **не привязана к конкретному типу данных и может хранить значения любого типа.** Тип переменной определяется из присвоенных ей данных и **может изменяться при каждом новом присваивании.**

## Блок кода

**Блок кода** — это группа инструкций, объедененных **логически** и **синтаксически**, и выполняющихся как **единое целое** в определённом контексте программы.

## Управляющая конструкция

**Управляющая конструкция** — это конструкция, которая **управляет порядком** выполнения кода (потоком выполнения программы).

## Итерация

**Итерация** — однократное выполнение тела цикла.

## Итерируемый объект

**Итерируемый объект** — это объект в языке Python, который реализует метод `__iter__()`, возвращающий итератор, либо метод `__getitem__()` с поддержкой целочисленной индексации, позволяющий последовательно получать свои элементы.

## Структура данных

**Структура данных (data structure)** — программная единица, позволяющая **хранить и обрабатывать множество** однотипных и/или логически связанных **данных**.

## Индексация элементов

**Индекс** — **порядковый номер** элемента, находящегося в **составе структуры данных**.

В Python **индексация начинается с** `0`. То есть первый элемент имеет индекс `[0]`, второй `[1]` и т.д.

## Метод

**Метод** — это **функция**, определённая **внутри класса**. При вызове через объект она **автоматически получает этот объект первым аргументом**.

**Синтаксис вызова:** `object.method()`

```py
s = 'Python'
print(s.upper())

PYTHON
```

В примере выше мы вызываем метод `upper()` принадлежащий классу `str` на переменной `s`. В результате метод возвращает новую строку `PYTHON`.

## Система счисления

**Система счисления** — символический метод записи чисел, то есть представление чисел с помощью письменных знаков.

В Python целые числа можно записывать в **десятичной** (по умолчанию), **двоичной** (`0b`), **восьмеричной** (`0o`) и **шестнадцатеричной** (`0x`) системах счисления. Основание системы указывает, сколько цифр используется для записи чисел (2, 8, 10, 16).

## Синглтон

**Синглтон** — объект, существующий в единственном экземпляре.

## Хешируемый объект

**Хешируемый объект** — это объект, для которого Python может вычислить **уникальный числовой код — хеш**.

**Хеш** — целое число, однозначно характеризующее содержимое значения.

Хеш позволяет быстро находить объекты в таких структурах, как [множества(set)](sets.md#что-такое-множество) или ключи [словарей(dict)](dicts.md#что-такое-словарь).
