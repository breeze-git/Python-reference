# Глоссарий

## Динамическая типизация

**Python** — язык с **динамической типизацией**. Это означает, что переменная **не привязана к конкретному типу данных и может хранить значения любого типа.** Тип переменной определяется из присвоенных ей данных и **может изменяться при каждом новом присваивании.**

## Блок кода

**Блок кода** — это группа инструкций, объедененных **логически** и **синтаксически**, и выполняющихся как **единое целое** в определённом контексте программы.

## Управляющая конструкция

**Управляющая конструкция** — это конструкция, которая **управляет порядком** выполнения кода (потоком выполнения программы).

## Итерация

**Итерация** — однократное выполнение тела цикла.

## Итератор

**Итератор** — это объект в Python, который реализует **протокол итерации**, то есть умеет **последовательно возвращать элементы из некоторой коллекции или последовательности**.

**Особенности:**

- итераторы можно обойти циклом `for`;
- итератор можно преобразовать в список или кортеж, с помощью функций `list()` и `tuple()`;
- итератор можно распаковать с помощью оператора распаковки `*`.

## Итерируемый объект

**Итерируемый объект** — это объект в языке Python, который реализует метод `__iter__()`, возвращающий итератор, либо метод `__getitem__()` с поддержкой целочисленной индексации, позволяющий последовательно получать свои элементы.

## Структура данных

**Структура данных (data structure)** — это **способ организации и хранения данных в памяти**, определяющий,
как данные расположены и какие операции над ними можно выполнять эффективно.

## Коллекция

**Коллекция** — это структура данных, предназначенная **для хранения, организации и обработки набора значений**.

## Индексация элементов

**Индекс** — **порядковый номер** элемента, находящегося в **составе структуры данных**.

В Python **индексация начинается с** `0`. То есть первый элемент имеет индекс `[0]`, второй `[1]` и т.д.

## Метод

**Метод** — это **функция**, определённая **внутри класса**. При вызове через объект она **автоматически получает этот объект первым аргументом**.

**Синтаксис вызова:** `object.method()`

```py
s = 'Python'
print(s.upper())

PYTHON
```

В примере выше мы вызываем метод `upper()` принадлежащий классу `str` на переменной `s`. В результате метод возвращает новую строку `PYTHON`.

## Система счисления

**Система счисления** — символический метод записи чисел, то есть представление чисел с помощью письменных знаков.

В Python целые числа можно записывать в **десятичной** (по умолчанию), **двоичной** (`0b`), **восьмеричной** (`0o`) и **шестнадцатеричной** (`0x`) системах счисления. Основание системы указывает, сколько цифр используется для записи чисел (2, 8, 10, 16).

## Синглтон

**Синглтон** — объект, существующий в единственном экземпляре.

## Хешируемый объект

**Хешируемый объект** — это объект, для которого Python может вычислить **уникальный числовой код — хеш**.

**Хеш** — целое число, однозначно характеризующее содержимое значения.

Хеш позволяет быстро находить объекты в таких структурах, как [множества(set)](sets.md#что-такое-множество) или ключи [словарей(dict)](dicts.md#что-такое-словарь).

## Поверхностная копия

**Поверхностная копия** — копия какого либо объекта без **полного** копирования вложенных коллекций, если таковые имеются. То есть, если **исходный объект содержит вложенные изменяемые коллекции** (например, списки или словари), то в копии **будут ссылки на те же самые вложенные объекты, а не их независимые копии.**

```py
lst = [1, 2, 3, 4, ['one', 'two']]
print(lst)

lst_copy = lst.copy()
lst[4][0] = 'three'

print(lst, lst_copy, sep = '\n')

[1, 2, 3, 4, ['three', 'two']] # оригинальный список изменился
[1, 2, 3, 4, ['three', 'two']]

```

## Парадигма программирования

**Парадигма** – устоявшаяся система научных взглядов, в рамках которой ведутся исследования (Т. Кун).

**Парадигма программирования** — совокупность идей и понятий, определяющих стиль написания компьютерных программ.

## Мультипарадигменный язык

**Мультипарадигменный язык** — это язык программирования, который поддерживает несколько парадигм программирования, позволяя разработчику выбирать наиболее подходящий стиль (или комбинировать их) для решения конкретной задачи.

Python - мультипарадигменный язык программирования.

## Компаратор

Компаратор - функция, определяющая условия сравнения элементов.

## Агрегация результата

**Агрегация результата** — формирование одного результирующего значения при комбинации элементов с использованием аргумента-аккумулятора.
