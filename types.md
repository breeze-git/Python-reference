# Встроенные типы данных

## Целые числа

**Integer (int)** — целочисленный тип данных. Его значением являются любые целые числа, по умолчанию в десятичной системе.

Но Python также поддерживает числа в **двоичной**, **восьмеричной** и **шестнадцатиричной** <a href="glossary.md#number-system">системах счисления</a>. Для указания, что число представляет двоичную, восьмеричную и шестнадцатеричную систему перед числом **ставится префикс `0b`, `0o` и `0x` соответственно.**

Явно указанное численное значение в коде программы называется **целочисленным литералом**.

Для удобного чтения чисел можно использовать символ **подчеркивания** — `13_000_000`.

---

## Вещественные числа

**Float (float)** — тип данных, значением которого являются числа с плавающей точкой в диапозоне `от ≈ 2.2 × 10⁻³⁰⁸ до ≈ 1.8 × 10³⁰⁸`.

Число float может иметь только 18 значимых символов, если число слишком большое его можно записывать, используя **экспоненту**. Число после экспоненты указывает степень числа 10, на которое надо умножить основное число, например:

```py
floatExp1 = 3.9e3 # все равно что 3.9 * 10³
print(floatExp1)
3900.0 

floatExp2 = 3.9e-3 # все равно что 3.9 * 10⁻³
print(floatExp2)
0.0039
```

---

## Комплексные числа

**<b id="#complex-numbers">Complex (complex)</b>** — тип данных, значением которого являются **комплексные числа** в формате `real + imag j`, где

- real – вещественная часть комплексного числа;
- imag – мнимая часть комплексного числа, которая завершается символом j или J.

```py
num = 1+2j
print(num)

(1+2j)
```

Обратите внимание: в Python используется буква `j`, а не `i`.

Комплексные числа можно создать с помощью **литерала** или **функции `complex()`**, которая принимает два аргумента: вещественную и мнимую часть числа, либо строковое представление числа.

```py
z1 = complex(1, 2)
z2 = complex('3+4j')

print(z1, z2, sep = '\n')

(1+2j)
(3+4j)
```

Для получения действительной и мнимой частей комплексного числа используются свойства `real` и `imag`:

```py
z = complex('3+4j')

print(z.real, z.imag)

3.0 4.0
```

**Свойства типа данных `Complex`:**

- Тип данных `complex` отлично интегрирован в язык Python. С `complex` числами работают все привычные операции: сложение, вычитание, умножение, деление, возведение в степень.

- Python позволяет совершать арифметические операции над `complex` и целыми числами (миксовать `complex`, `int`, `float`).

- В стандартной библиотеке Python есть модуль <a href="modules.md#cmath-module">`cmath`</a> включающий дополнительные функции для работы с комплексными числами.

---

**Метод `z.conjugate()`** — возвращает сопряженное комлексное число для числа `z`.

```py
z = 3+4j

print(z.conjugate())

(3-4j)
```

---

**Функция `abs()`** — позволяет получить модуль комплексного числа.

```py
z = 3+4j

print(abs(z))

5.0
```

## Логический тип

**Boolean (bool)** — логический тип данных. Имеет только два значения — `True(истина)` и `False(ложь)`. Данные логические значения можно трактовать как числа. Значению `True` соответствует число `1`, значению `False` соответствует `0`.

## Тип NoneType

**<b id="#nonetype">NoneType</b>** — тип данных, **единственным значением которого является специальное значение `None`**, обозначающее `ничего`, `отсутствие данных`.

Для проверки значения переменной на `None`, рекомендуется использовать оператор `is`. Для встроенных типов поведение `is` и `==` абсолютно одинаково, однако с пользовательскими типами могут возникнуть проблемы, так как в Python есть возможность переопределения операторов сравнения в пользовательских типах.

**Примечания:**

- Сравнение `None` с любым объектом, отличным от `None`, дает значение `False`.
- Тип `NoneType` — **синглтон** (объект, существующий в единственном экземпляре).

# Динамическая типизация

Python — язык с **динамической типизацией**. Это означает, что переменная **не привязана к конкретному типу данных и может хранить значения любого типа.** Тип переменной определяется из присвоенных ей данных и может изменяться при каждом новом присваивании.

C помощью встроенной функции `type()` можно динамически узнать текущий тип переменной.

```py
num = 10
print(type(num))
<class 'int'>
```

# Преобразование типов

**Функция `int()`** — используется для преобразования к целочисленному типу данных `integer`.

---

**Функция `str()`** — используется для преобразования в строковой тип данных `string`.

---

**Функция `float()`** — используется для преобразования в число с плавающей точкой.

**Неявное преобразование** — любое целое число (`тип int`) можно использовать там, где ожидается число с плавающей точкой (`тип float`), поскольку **при необходимости Python автоматически преобразует** целые числа в числа с плавающей точкой.

**Явное преобразование** — число с плавающей точкой **нельзя неявно преобразовать** в целое число. Для такого преобразования **необходимо использовать явное преобразование** с помощью команды `int()`.

**Преобразование чисел** с плавающей точкой в целое производится **с округлением в сторону нуля**, то есть `int(1.7) = 1`, `int(-1.7) = -1`.

Если строка не является **целым числом**, то при преобразовании возникнет **ошибка**.

---

**Функция `list()`** — используется для преобразования в список.
