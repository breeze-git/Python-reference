# Основные типы данных

**Boolean (bool)** — логический тип данных. Имеет только два значения — `True(истина)` и `False(ложь)`.

---

**Integer (int)** — целочисленный тип данных. Его значением являются любые целые числа, по умолчанию в десятичной системе.

Но Python также поддерживает числа в **двоичной**, **восьмеричной** и **шестнадцатиричной** системах. Для указания, что число представляет двоичную, восьмеричную и шестнадцатеричную систему перед числом **ставится префикс `0b`, `0o` и `0x` соответственно.**

Явно указанное численное значение в коде программы называется **целочисленным литералом**.

Для удобного чтения чисел можно использовать символ **подчеркивания** — `13_000_000`.

---

**Float (float)** — тип данных, значением которого являются числа с плавающей точкой в диапозоне `от ≈ 2.2 × 10⁻³⁰⁸ до ≈ 1.8 × 10³⁰⁸`.

Число float может иметь только 18 значимых символов, если число слишком большое его можно записывать, используя **экспоненту**. Число после экспоненты указывает степень числа 10, на которое надо умножить основное число, например:

```py
floatExp1 = 3.9e3 # все равно что 3.9 * 10³
print(floatExp1)
3900.0 

floatExp2 = 3.9e-3 # все равно что 3.9 * 10⁻³
print(floatExp2)
0.0039
```

---

**Complex (complex)** — тип данных, значением которого являются комплексные числа в формате `вещественная_часть+мнимая_частьj` — после мнимой части указывается суффикс `j`.

```py
complexNumber = 1+2j
print(complexNumber)
(1+2j)
```

---

**String (str)** — тип данных, значением которого являются **строки** — произвольные последовательности символов. Значения типа String могут содержать любые символы, включая буквы, цифры, знаки препинания и специальные символы, и обычно заключаются в **одинарные(' ')** или **двойные(" ")** кавычки. В Python строки представляют собой набор символов **в кодировке Unicode.**

**Для создания строковой переменной (литерала)** нужно заключить необходимый текст в **кавычки**.

```py
longText = "The evening sunset painted the sky" \ 
                "with shades of orange and pink."
print(longText)
The evening sunset painted the sky with shades of orange and pink.
```

---

Чтобы записать в переменнную многострочный текст, нужно заключить его в **тройные кавычки (''' ''').**

---

Для управления выводом текста и форматирования строк в Python используются **управляющие последовательности**. Они позволяют вставлять специальные символы, которые изменяют отображение строки, например, для переноса строки — `\n`, табуляции — `\t` или других эффектов.

---

**Чтобы отключить интерпретацию управляющих последовательностей** в строке, перед ней ставится **символ `r`**.

---

Чтобы вставить в строку **значения других переменных**, необходимо разместить название переменных в фигурных скобках `{ }`, и перед всей строкой поставить **символ `f`**.

```py
print(f' First Name: {firstName} \n Last Name: {lastName}')
```

---

**Конкатенация** — операция сложения строк.

```py
print('Py' + 'thon')
Python
```

---

**Оператор умножения строки** на число (repetition) - позволяет умножить строку на целое неотрицательное число.

```py
print('Python' * 3)
PythonPythonPython
```

**При умножении строки на 0 получается пустая строка — `''`**.

---

# Динамическая типизация

Python — язык с **динамической типизацией**. Это означает, что переменная **не привязана к конкретному типу данных и может хранить значения любого типа.** Тип переменной определяется из присвоенных ей данных и может изменяться при каждом новом присваивании.

C помощью встроенной функции `type()` можно динамически узнать текущий тип переменной.

```py
num = 10
print(type(num))
<class 'int'>
```

# Преобразования типов

**Функция `int()`** — используется для преобразования к целочисленному типу данных `integer`.

---

**Функция `str()`** — используется для преобразования в строковой тип данных `string`.

---

**Функция `float()`** — используется для преобразования в число с плавающей точкой.

**Неявное преобразование** — любое целое число (`тип int`) можно использовать там, где ожидается число с плавающей точкой (`тип float`), поскольку **при необходимости Python автоматически преобразует** целые числа в числа с плавающей точкой.

**Явное преобразование** — число с плавающей точкой **нельзя неявно преобразовать** в целое число. Для такого преобразования **необходимо использовать явное преобразование** с помощью команды `int()`.

**Преобразование чисел** с плавающей точкой в целое производится **с округлением в сторону нуля**, то есть `int(1.7) = 1`, `int(-1.7) = -1`.

Если строка не является **целым числом**, то при преобразовании возникнет **ошибка**.
